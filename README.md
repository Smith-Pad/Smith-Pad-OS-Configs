# Smith-Pad-OS-Configs
Configs for Smith-Pad-OS

## Dependencies 

> PHP

> Arch Linux 

> git



## Installing the Smith-Pad-OS-Configs



```shell

## Step 1
sudo su


## Step 2 
cd


## Step 3 
git clone https://github.com/Smith-Pad/Smith-Pad-OS-Configs --recursive


## Step 4
cd Smith-Pad-OS-Configs 

## Step 5 

sh configure.sh

```
